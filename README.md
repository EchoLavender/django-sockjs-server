# Django SockJS-server 

Simple sockjs server for django.

## Requirements:

* sockjs-tornado >= 1.0.0
* pika >= 0.9.12
* django >= 1.4

## Installation:
```
pip install django-sockjs-server
```

Add django-sockjs-server to your INSTALLED_APPS in settings.py

```
  INSTALLED_APPS = (
       ...
       'django_sockjs_server',
       ...
  )
  ```
